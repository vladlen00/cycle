# Mini App: Цикл

Telegram Mini App для @relax2000_bot, женский трекер менструального цикла. Часть Студии Ирены Пол. Минимально-расширяемая v1.

## КРИТИЧЕСКОЕ ПРАВИЛО: соседние папки только для чтения

Правило универсальное: применяется ко всем существующим соседним папкам и ко всем новым, которые могут появиться. Конкретный список имён не ведём, чтобы он не устаревал (часть папок может находиться на OneDrive или существовать только в GitHub - проверяй фактически на момент работы).

## Архитектура CRUD

Запись и чтение данных в таблицу cycles идёт через отдельную Edge Function cycles-api (НЕ через прямой REST к /rest/v1/).

Поток:
- Frontend → POST /functions/v1/cycles-api с body { action, payload, initData }
- Edge Function валидирует HMAC initData по стандарту Telegram Mini App
- Извлекает Telegram user_id из валидированного initData
- Выполняет CRUD к cycles через service_role (обходит RLS)
- Возвращает результат

Поддерживаемые actions: list, create, update, delete.

user_id в таблице cycles = Telegram user.id (число), не случайный localStorage ID.

Anon key из Supabase в коде фронта НЕ используется (в отличие от biohack). Все запросы к данным идут только через cycles-api.

Edge Function verify-access остаётся отдельной и не трогается, она проверяет членство в платном канале и пускает в мини-аппу.

## Функционал v1 (минимум)
1. Главный экран: круговой индикатор с днём цикла, фазой, прогнозом следующей менструации

3. Календарь: последние 3 месяца, цветные точки на менструациях
4. История: последние 6 циклов с длительностью и отклонением от среднего
5. Редактирование записей

## Логика расчёта (вынести в cycleCalc.js)
- Лютеиновая фаза всегда ~14 дней
- День овуляции = средняя длина цикла - 14
- Менструация: дни 1..menstruation_length_days (по умолчанию 5)
- Фолликулярная: между концом менструации и днём овуляции
- Овуляция: окно из 3 дней вокруг расчётного дня
- Лютеиновая: после овуляции до следующей менструации

Средняя длина цикла = среднее последних 6 записей, фильтр аномалий 21-45 дней. Дефолт 28 если данных мало.

## Палитра
Тёплая женская: пастельный розовый, кремовый, бежевый.
- Менструация: мягкий красный (не ярко)
- Фолликулярная: мятный
- Овуляция: золотистый
- Лютеиновая: лавандовый

ВАЖНО: не использовать длинные тире нигде, только короткие дефисы. На "ты". Тёплый, поддерживающий тон.
